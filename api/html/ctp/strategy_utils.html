Return a logger with two handlers, one for console and one for file. :param folder: string, the folder to store the file log. :param has_console: boolean, True if the console logger is enabled. :param has_file: boolean, True if the file logger is enabled. :param log_file_name: string, the name of the log file. before writing them to file.
