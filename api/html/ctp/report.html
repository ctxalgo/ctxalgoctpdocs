get_positions(instrument_id=None, period=None, start_time=None, end_time=None, merge=False)¶

Get instrument positions. :param instrument_id: string, instrument id. If None, return positions for all traded instruments

in this report. If not None, only return positions for the given instrument.
Parameters:
  • period – Periodicity, the periodicity of the commission. If None, use self.default_period.
  • start_time – datetime, the start time (inclusive) to get the commission. If None, start from the first trade.
  • end_time – datetime, the end time (inclusive) to get the commission. If None, ends at the last trade.
  • merge – boolean, if False, no merge will happen.
Returns:

dict{string: dict{string: OHLC}}, meaning dict{instrument_id: dict{position_direction: position_OHLC}} Outer keys are instrument ids. Inner keys are position direction, either ‘long’ or ‘short’. Inner values are OHLC objects whose closes series represent the position information.

get_transactions(instrument_id=None, direction=None, start_time=None, end_time=None)¶

Get transactions for an instrument. :param instrument_id: string, instrument id. If None, return positions for all traded instruments

in this report. If not None, only return positions for the given instrument.
Parameters:
  • direction – string, can be ‘long’, ‘short’ or None. Return only transactions in the given direction. If None, return transactions in both long and short directions.
  • start_time – datetime, the start time (inclusive) to get the commission. If None, start from the first trade.
  • end_time – datetime, the end time (inclusive) to get the commission. If None, ends at the last trade.
Returns:

dict{string: TransactionList}, the list of transactions. Keys are instrument ids, values are transactions for that instrument.

get_transactions_in_bars(transactions, ohlc, bars, reversed=False)¶

Select the transactions which starts within the specified bars. In other words, a transaction is selected from transactions and put into the result if and only if its start time is within a ohlc bar whose 0-based bar indexes are in bars. :param transactions: [CtpTransaction], the list of transactions to choose from. :param ohlc: OHLC, the ohlc object. :param bars: [int], the 0-based bar indexes. Only transactions whose start time is

within these bars are selected.
Parameters:reversed – boolean, if True, perform the reversed selection. That is, select a transaction if its start time is not in any of bars.
Returns:[CtpTransaction], the selected transactions.
