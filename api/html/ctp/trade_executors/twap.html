class ctxalgoctp.ctp.trade_executors.twap.StateMachine(instrument_id, strategy, duration, number_of_slices, minimum_slice_length)¶

Bases: object

Parameters:
  • instrument_id – string, instrument id. A state machine is for a single instrument only, to support trading multiple instruments, different state machines are created.
  • strategy – AbstractStrategy, the strategy.
  • duration – timedelta, the duration of a twap task.
  • number_of_slices – int, the number of slices in a twap task. This means we will split duration into number_of_slices parts and trade a certain amount of volumes in each part, that’s is the twap semantics.
  • minimum_slice_length – int, the minimum length (in second) of each slice in a twap operation.
