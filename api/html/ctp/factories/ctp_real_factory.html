class ctxalgoctp.ctp.factories.ctp_real_factory.CtpRealFactory(account, security_company=None, data_feed=None, account_initial_capital=1000000.0, dominant_provider=None)¶

Bases: ctxalgoctp.ctp.factories.abstract_ctp_factory.AbstractCtpFactory

This factory returns real CTP market and trading classes.

Initialize current. :param account: dict{string: object}, information of the trading account.

The dict must have the following keys: market_server, trading_server, user, password, broker_id. market_server: string, the server for market data. trading_server: string, the server for trading data. user: string, user name. password: string, password. broker_id: string, broker id.
Parameters:
  • security_company – string, security company name.
  • data_feed – AbstractDataFeed, the data feed.
  • account_initial_capital – float, the initial capital for the trading account. This is used to make sure the strategy report will use the current initial capital to calculate metrics.
