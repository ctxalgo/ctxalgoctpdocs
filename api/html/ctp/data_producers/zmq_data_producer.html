is_tick_needed()¶

Since in ZmqDataProducer, tick and bar data comes independently, we don’t need tick data if we only want to have bar data (which is different from CtpDataProducer, which requires tick data from CTP to generate bar data). We can set_is_tick_needed with False to reduce network traffic. :return: bool
