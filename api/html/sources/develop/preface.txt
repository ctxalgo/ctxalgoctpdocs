supervisor
===========

Supervisor_ is a client/server system that allows its users to control a number of processes on UNIX-like operating systems.

::

    sudo pip install supervisor

virtualenv
===========

virtualenv_ is a tool to create isolated Python environments.

::

    sudo pip install virtualenv

virtualenvwrapper
===================

virtualenvwrapper_ is a set of extensions to Ian Bickingâ€™s virtualenv tool. The extensions include wrappers for creating and deleting virtual environments and otherwise managing your development workflow, making it easier to work on more than one project at a time without introducing conflicts in their dependencies.

::

    sudo pip install virtualenvwrapper

Then make a virtualenv::

    mkvirtualenv <env_name>

and active the env::

    workon <env_name>

.. note::

    Will run well without above
